# KUCI Listener Tracker

## Overview

Tracks and plots the current listener count for [KUCI 88.9 FM Irvine](https://kuci.org), with data pulled from the [Icecast server](https://streamer.kuci.org).

## Stack

Uses Vite for frontend development, utilizing a Vue + TypeScript stack and Tailwind CSS for styling. Graph.js is used for visual data representation.
